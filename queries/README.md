# Example Queries

This folder contains hands-on KQL examples grouped by difficulty.

Structure:
- basic/ — quick, fundamental commands (take, where, project, summarize)
- medium/ — aggregations, binning, make-series
- complex/ — joins, windowing, time-series joins

Open a `.kql` file in the query editor (Azure Data Explorer or Kusto.Explorer) and run it. Many samples include a `datatable(...)` so they run without pre-existing tables.
