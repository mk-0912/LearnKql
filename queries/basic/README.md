# Basic KQL Examples

Files in this folder show common beginner queries.

- take_examples.kql — sample `take`, `where`, `project`, and `summarize` queries using `datatable` so you can run them immediately.

How to use
1. Open Azure Data Explorer web UI (https://dataexplorer.azure.com/) or Kusto.Explorer.
2. Paste the contents of the `.kql` file and run.
3. Modify the datatable or conditions to experiment.
